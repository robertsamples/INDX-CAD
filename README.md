# INDX-CAD
reverse engineered CAD from Prusa's printable model with internal geometry

<img width="828" height="548" alt="image" src="https://github.com/user-attachments/assets/7b6237ae-3244-415b-b21a-b2d3dfbaa57e" />

Reverse engineered geometry from Prusa's [printable model](https://www.printables.com/model/1761468-indx-toolhead) which is licensed under CC BY-SA 4.0. Originally uploaded to [printables](https://www.printables.com/model/1775408-indx-cad-conversion-internal-step-file/files). The model uploaded by prusa was meshed well enough for parametric geometry to be extracted for STEP conversion. Consider this a work in progress as filleted components (mostly injection molded parts) did not convert as well as CNCed parts.
