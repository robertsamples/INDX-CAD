# INDX-CAD
reverse engineered CAD from Prusa's printable model with internal geometry

<img width="828" height="548" alt="image" src="https://github.com/user-attachments/assets/7b6237ae-3244-415b-b21a-b2d3dfbaa57e" />

Reverse engineered geometry from Prusa's printable which was licensed under CC BY-SA 4.0 (original 3mf included here for archival purposes). Originally uploaded to printables but the original model and remixes were removed. The model uploaded by prusa was meshed well enough for parametric geometry to be extracted for STEP conversion. Consider this a work in progress as filleted components (mostly injection molded parts) did not convert as well as CNCed parts. I am still working on improving the geometry and am choosing to host it in a less visible space than usual 3DP hosting sites.
